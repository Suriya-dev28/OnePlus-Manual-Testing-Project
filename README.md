# OnePlus Manual Testing Project

## Project Overview
This project demonstrates manual testing of the OnePlus E-Commerce website. The objective is to validate core functionalities, identify defects, and ensure the application meets business requirements.

## Modules Tested
- Login
- Registration
- Product Search
- Product Details
- Add to Cart
- Checkout
- Payment

## Testing Types
- Functional Testing
- Smoke Testing
- Sanity Testing
- Regression Testing
- Positive Testing
- Negative Testing

## Deliverables
- Test Plan
- Test Scenarios
- Test Cases

## Tools Used
- Manual Testing
- MS Excel
- GitHub
